الوظيفة	الاختصار (Linux/XWin)
البحث في كل الملفات	Double Shift
البحث عن ملف	Ctrl + Shift + N
البحث عن رمز (Class/Function)	Ctrl + Shift + Alt + N
الانتقال إلى سطر معين	Ctrl + G
الانتقال إلى التعريف	Ctrl + B أو Ctrl + Click
العودة إلى الموقع السابق	Ctrl + Alt + ←
التقدم إلى الموقع التالي	Ctrl + Alt + →
فتح نافذة الأوامر (Run Anything)	Ctrl + Shift + A
2. التعديل على الكود
الوظيفة	الاختصار
تعليق/إلغاء التعليق	Ctrl + /
نسخ السطر الحالي	Ctrl + D
حذف السطر الحالي	Ctrl + Y
سحب السطر لأعلى/أسفل	Ctrl + Shift + ↑/↓
تنسيق الكود تلقائيًا	Ctrl + Alt + L
إعادة تسمية متغير/دالة	Shift + F6
إنشاء دالة/متغير سريعًا	Alt + Insert
إصلاح الأخطاء تلقائيًا	Alt + Enter
3. إدارة الملفات والنوافذ
الوظيفة	الاختصار
فتح ملف حديثًا	Ctrl + E
إغلاق التبويب الحالي	Ctrl + F4
فتح Terminal مدمج	Alt + F12
تكبير/تصغير المحرر	Ctrl + Shift + F12
تقسيم الشاشة أفقياً/رأسياً	Ctrl + Shift + [1/2/3]
4. إدارة Git والتصحيح (Debug)
الوظيفة	الاختصار
تنفيذ البرنامج (Run)	Shift + F10
تصحيح الأخطاء (Debug)	Shift + F9
التنقل بين الأخطاء	F2 / Shift + F2
فتح Git Commit	Ctrl + K
سحب التحديثات (Git Pull)	Ctrl + T

الوظيفة	الاختصار

إظهار التوثيق (Documentation)	Ctrl + Q
توسيع/تصغير الكتلة	Ctrl + +/-
إظهار بنية الملف (File Structure)	Ctrl + F12
